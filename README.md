# Amazon Clone

A simplified clone of the Amazon homepage using **HTML** and **CSS**.

## Features
- Basic homepage layout similar to Amazon.
- Responsive design for desktop and mobile.

## Tech Stack
- **HTML**
- **CSS**

## Installation
Clone the repository and open `index.html` in your browser.
